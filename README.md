# meteor-github-md

## Installation

```sh
$ meteor add pntbr:github-md
```

## How to use

First you can create a github repo :

```sh
$ meteor shell

> repo = new Repo('pointbar', 'repo');
```

and play with his methods :

```sh
> repo.isAble
true

> repo.getFile('', 'README.md')
{json}

>
```
